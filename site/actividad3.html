Laboratorio - Cómo crear un repositorio en GIT HUB


Para reforzar los conocimientos adquiridos debes realizar un taller practico para aplicar lo aprendido a continuacion sigue los siguientes pasos y crear tu propio repositorio.


Paso 1:


Ingresas a GitHub.com, y seleccionas la opcion de crear un nuevo repositorio.


Paso 2:


Se asigna el nombre que tu desees al repositorio



Paso 3:


Ingresa una descripcion y el tipo de privacidad para ti repositorio


Paso 4:


Por ultimo marcamos la opcion Initialize this repository with a README y seleccionamos la opcion de crear repositorio.


Nota:


Recuerda buscar en mas fuentes y practica en la creacion de tus paginas para diseñar mas paginas increibles.
